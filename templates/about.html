About the Project

Learn more about our California House Pricing Prediction system

Project Overview

This web application uses machine learning to predict house prices in California based on various property features. The system is built using Flask and incorporates a Random Forest regression model trained on the California Housing Dataset.

Machine Learning Model
  • Algorithm: Random Forest Regressor
  • Training Data: California Housing Dataset
  • Features: 9 property characteristics
  • Preprocessing: Automated pipeline
  • Validation: Stratified cross-validation
Technical Stack
  • Backend: Flask (Python)
  • ML Library: Scikit-learn
  • Data Processing: Pandas & NumPy
  • Frontend: Bootstrap 5
  • Model Storage: Joblib
Features Used for Prediction
Location Features:
  • Longitude
  • Latitude
  • Ocean Proximity
Property Features:
  • Housing Median Age
  • Total Rooms
  • Total Bedrooms
Demographic Features:
  • Population
  • Households
  • Median Income
How to Use
Single Prediction:
  1. Navigate to the "Predict" page
  2. Fill in all required property details
  3. Click "Predict Price" to get instant results
  4. View the predicted price and confidence score
Batch Prediction:
  1. Prepare a CSV file with property data
  2. Upload the file using the "Upload" page
  3. Wait for processing to complete
  4. Download the results with predictions
Important Notes
  • Predictions are based on historical California housing data
  • Results should be used as estimates, not exact values
  • Market conditions may affect actual prices
  • For CSV uploads, ensure all required columns are present
  • Ocean proximity values must match the specified options
